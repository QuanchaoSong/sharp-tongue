### How to Run

Open Terminal and run the following commands:

```bash
git clone https://github.com/QuanchaoSong/sharp-tongue-nextjs.git
cd sharp-tongue-nextjs/
npm install
npm run dev
```

Then open "http://localhost:3000" in web browser.
